Source Code Conventions



Layers Overview

  • Base, general purpose foundation classes for 3d applications
  • Engine, 3d game engine
  • Game, application-specific code


Layers

Base Modules:
  • 2D, font rendering
  • 3D, 3d api abstraction (3d backend)
  • Audio, audio backend (background music and sample playback)
  • App, 3d application base functionality
  • Input, input backend (gamepad, mouse, keyboard)
  • IO, I/O related functionality
  • Math, mathematical classes
  • Util, utility classes for different purposes
  • Reflection, reflection functionality
  • Scripting, scripting interface
  • System, file and message dialogs
Engine Modules:
  • 2D overlays
  • App, engine application base functionality
  • Geometry, meshes, particle systems, terrains and subdivision surfaces
  • Scenegraph,transformation hierarchy, hier. bounding volume tree
  • Materials, shaders and textures
  • Rendering, state sorting, culling
  • Physics, physics API binding
  • Animation, kinematic animations
  • I3D, i3d loader


Internal module dependencies

  • App: Geometry, Scenegraph, Materials, Rendering, Physics, Animation, I3D
  • Geometry: 3D, Math, Util
  • Scenegraph: Geometry, Materials, IO, Math, Util
  • Materials: 3D, IO, Math, Util
  • Rendering: Geometry, Materials, Scenegraph, 3D, App, Input, IO, Util
  • Physics: Geometry, Scenegraph, Math, Util
  • Animation: Scenegraph, Math, Util
  • I3D: Geometry, Materials, Scenegraph, IO, Math, Util


External dependencies (libraries)

  • Expat, XML SAX Parser
  • TinyXML, XML DOM Parser
  • DevIL, image library
  • LUA, script language
  • SDL, Simple Directmedia library
  • Cg, NVIDIA Cg shader language
  • OpenAL, Cross-Platform 3D Audio
  • Minizip, Zip and UnZip library
